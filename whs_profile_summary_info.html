Please fill out all fileds that has GREEN colored background.
The fields with DARKER GREEN has predefined values available, please use the drop down menu.
All fields are mandatory.
Warehouse country, city and code fields are setup as dependent drop down menus.  First select the country and continue with the city, finaly select the desired warehouse code.  If warehouse code is not listed, please see the Info - WHS Codes sheet. Than specify the desired WHS code in the cell G6 (overwrite  text).  From the first dropdown menu select  "Country_custom", from the city drop down menu select  "City_custom" - the last drop down menu than should contain the value in the cell G6. For proper functionality, please first enter the value, than use the drop down selection.
Length Unit Field is equipped with dropdown menu. Only authorised values are offered. Please select one of the following units: Inch, Meter, Centimeter, Decimeter, Kilometer, Mile
Weight Unit Field is equipped with dropdown menu. Please select one of the following units: Tonne, Kilogram, Gram, Pounds.
Volume Unit Field is equipped with with dropdown menu. Please select one of the following units: Cubiccentimeter, Hectoliters , Liter, Cubicmeter, Mililiter, CubicMilimeter.
Default lcoation must be one location  from the locactions specified on sheet Warehouse location Mgmt in column B.
The drop down menu contains only values found in the WHS locations specified on sheet  Warehouse location Mgmt in column B lines between 18 - 3000.
Default receiving location - is prefilled in Receive Onhand screen. Also should be pre-filled when receiving HB  (both scanner and desktop).
Default unloading location - is automatically set when consolidated packages are received inside container (both scanner and desktop).
Warehouse timezone field is equipped with data validation function. Only authorised values are offered. Please select reqion first, than select time zone. All date and time values within the application are converted into this time (unless not explicitly stated, e.g. UTC time column in inbound log)
IATA location field which is automatically set as Origin value when Creating HB or Truck. (Country code + IATA of the city). This filed is autofilled, but can be overwritten if needed.
Warehouse Local Language field - is equipped with dropdown menu. Please select one of the offered values. Be kindly informed that this feature is under development and not yet implmented.
Hide consignee on HB label - Printing of consignee name in the additional info section on the HB label: default value is 0 , info will be printed .
Default Custom Clearance: How to treat a shipment when no customs info is available in the interface data from NFE, default value is 0  shipment will be on hold in FSI .
